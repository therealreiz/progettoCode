Lista comandi HTML:

!DOCTYPE>:

Definisce il tipo di documento HTML e la versione utilizzata.

html

Definisce il contenuto HTML di una pagina web.

head:

Contiene metadati e informazioni di intestazione come il titolo della pagina, collegamenti ai fogli di stile CSS e script JavaScript.

title:

: Definisce il titolo della pagina, che appare nella barra del titolo del browser.

body:

: Contiene il contenuto principale della pagina che viene visualizzato dall'utente, come testo, immagini, video, ecc.

Intestazioni (Heading):

I tag h1 - h6 definiscono intestazioni di diversi livelli, dove h1 è il più grande e h6 il più piccolo. Sono utilizzati per organizzare il testo e creare una struttura gerarchica nella pagina.

P

: Definisce un paragrafo di testo.

a

Crea un collegamento ipertestuale (link) a un'altra pagina web o a una risorsa.

img

Inserisce un'immagine nella pagina web. Liste:

ul:

Definisce una lista non ordinata.

ol:

Definisce una lista ordinata.

li:

Definisce un elemento della lista.

table:

Definisce una tabella.

tr:

Definisce una riga di una tabella.

th:

Definisce una cella di intestazione di una tabella.

td:

Definisce una cella di dati di una tabella.

form:

Crea un modulo per la raccolta di dati dall'utente.

input:

Definisce un campo di input all'interno di un modulo.

textarea:

Crea un campo di testo multiriga all'interno di un modulo.

button:

Crea un pulsante cliccabile.

div:

Un contenitore generico che può essere utilizzato per raggruppare e organizzare altri elementi HTML.

span:

Un contenitore generico utilizzato per applicare stili o manipolare il testo all'interno di una pagina. Questi sono solo alcuni dei numerosi tag HTML disponibili. Ogni tag ha uno scopo specifico e può essere utilizzato per costruire layout e funzionalità diverse all'interno di una pagina web.
